# AI FAQ Chatbot 🤖

This is an AI-powered FAQ chatbot built using NLP and Streamlit.

## Features
- Answer questions from dataset
- NLP preprocessing
- TF-IDF + Cosine Similarity
- Streamlit Web Interface

## Tech Stack
- Python
- Pandas
- NLTK
- Scikit-learn
- Streamlit

## How to Run
streamlit run app.py
